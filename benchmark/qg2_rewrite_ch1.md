# 第一章 断肠散

胃里像灌了一壶滚油。

这是他睁开眼睛之前唯一确定的事。

陆九针——不，他还不知道自己叫这个名字——只觉得五脏六腑被人攥成一团，拧，再拧，像急诊室里那些误服强酸的病人描述的那样。他见过太多次了。只不过这回，疼的是自己。

苏原。他叫苏原。二十八岁，市第一人民医院急诊科第三年。

三秒前他还在抢救台前。

眼前却是一顶绣着暗纹的床帐。

烛火跳了一下，油烟味混着药苦味钻进鼻腔。不是医院。不是消毒水和碘伏的气味。是草药。是劣质灯油。是——

胃又痉挛了一下，他本能地侧过身，呕出一口黑血。

"少爷！"

一双粗糙的手扶住他的肩。是个老头，花白头发扎得整整齐齐，一张脸皱得像晒干的核桃。眼睛却亮得厉害，里面全是惊慌。

"少爷你醒了？老天爷……福叔去叫大夫，你撑住——"

他抓住老人的手腕。

不是有意的。是肌肉记忆。急诊科三年，病人要跑你第一反应就是摁住。他摁住了这只手腕，拇指恰好压在桡动脉上。

脉搏——不，不对，这是别人的手腕。

他低头看自己的手。

细。白。指节分明。不是他的手。他的手该有一道旧疤，右手虎口，实习时缝合针扎的。

没有疤。

胃里第二波绞痛炸开。这次比第一波狠三倍。他整个人弓成虾米，冷汗瞬间湿透了后背。

"少爷！少爷你怎么了！"老人的声音在发抖。

他张了张嘴。喉咙像被砂纸打磨过，但他必须问。

"我……吃了什么？"

"晚饭后大少爷送来的燕窝粥，少爷你——"

"多久了？"

"什么？"

"我吃了那碗粥，到现在，多久了？"

福叔愣了一瞬："约莫……两个时辰。"

四个小时。

他不知道这里的度量衡是不是和他认知中的一样。但身体的反应不会骗人。四小时的毒物代谢窗口，内脏还没开始出血——至少呕出来的黑血量不大，更像是胃黏膜灼伤而非穿孔——这不是砒霜，不是鹤顶红。

是什么？

他闭上眼，把注意力从胃里的灼痛拉开，转向全身。颈椎。胸椎。腰椎。四肢。

感觉在。痛觉在。但——

他试着握拳。

力气像从指缝间漏出去的沙。明明大脑在下指令，肌肉却只回应了一成。

不是麻痹。不是神经阻断。更像是……某条看不见的通路被堵死了。

一个他不认识的词忽然从这具身体的记忆深处浮上来——

经脉。

记忆不是他的。零碎的，像打碎的镜子。他看见一个中年男人的背影，听见有人叫"九针"，闻到墨汁和药草混在一起的气味。陆家。庶子。二十二岁。

然后就是那碗燕窝粥。

堂兄笑着端来的。"九针弟弟近日辛苦，特地吩咐厨房熬的。"

他没时间理清这些。胃里的灼烧每一秒都在加剧，再不处理，四小时就会变成不可逆。

"福叔。"他的声音意外地稳，"这屋里有银针吗？"

老人一怔："少爷平日用的那套……在柜子里。"

"拿来。再烧一壶热水。要滚开的。"

福叔张了张嘴，似乎想说什么，但看见他的眼神后，什么也没问，转身就走。

动作比年轻人还利索。

陆九针——他决定暂时接受这个名字——撑着床沿坐起来。房间晃了一下。他等眩晕过去，开始检查自己的身体。

左手搭上右手腕。

脉象沉涩，几乎摸不到。换个位置，寸关尺三部全是同样的衰微。按照他在急诊科的判断，这脉象意味着循环系统接近衰竭。但他还能坐着说话，意识清醒——

这具身体的"循环系统"和他理解的不一样。

那些碎片记忆告诉他，这叫"经脉"。十二正经，奇经八脉，是这个世界的武者运行内力的通路。而此刻，他能感知到的通路……

十成只剩一成。

其余九成像被烧熔的电线，信号全断了。

断肠散。

这三个字从碎片记忆里跳出来，带着寒意。原主知道这个名字。陆九针知道这是什么——一种慢性剧毒，四个时辰内灼烧经脉，六个时辰后毒入骨髓，届时大罗金仙也救不回来。

他还有两个时辰。

福叔端着一只黄杨木盒和一壶热水回来时，手还在抖。

"少爷，你真的不要叫大夫？"

"来不及了。"他打开木盒。

三十六根银针，长短不一，按尺寸排列在绒布上。银质上等，针尖在烛光下泛着冷芒。他拿起最细的一根，在烛火上过了一遍——

不是消毒。这里没有酒精和碘伏。但高温聊胜于无。

他把热水倒进铜盆，将双手浸入，烫得皮肤发红。血液循环加速了一点。手指上的感觉回来了一些。

然后他拿起第一根针。

"福叔，帮我把后背的衣服解开。"

老人沉默地上前，替他褪去中衣。

夜风从窗缝灌进来，扑在裸露的脊背上，凉得他打了个激灵。但下一秒，他把所有注意力收拢到针尖上。

第一针，落在左手臂弯。不是他认知中的"曲池穴"——但位置重合度极高。针尖刺入皮肤的瞬间，他感觉到一股微弱的热流，像是冻住的水管里终于渗出一丝水。

经脉的反馈。

他前世二十八年从未感受过的东西。

但他没时间惊奇。热流微弱到几乎可以忽略，而毒素正在以远快于热流的速度继续蚕食。

第二针，第三针，第四针。

他在急诊科缝过最复杂的伤口是一个被玻璃割伤颈动脉的醉汉，八层组织逐层缝合，手不能抖半分。此刻他做的事比那还难——他要在一套自己从未真正使用过的"经脉系统"里找到还活着的通路，用银针刺激它们，逼出残余的内力去包裹毒素，阻止扩散。

像在一栋着火的大楼里，找到还没烧断的水管，拧开阀门。

第十二针落下时，他听见自己的呼吸变粗了。

手在抖。

不是紧张。是毒素开始攻心。胸口像被一只无形的手攥住，每一次心跳都带着钝痛。

"少爷……"福叔蹲在旁边，声音哑了。老人看不懂他在做什么，但他看得懂那张年轻的脸上全是冷汗，看得懂那双手越来越不稳。

"没事。"他说。嘴唇发白，但嘴角居然弯了一下，"急诊科抢救过比这更难的。"

福叔不知道"急诊科"是什么，但他没问。

第十六针。

第二十针。

身体已经开始发冷。不是夜风的冷。是血液流速降低、末梢供血不足的冷。他太熟悉这种冷了——失血性休克的病人就是这个温度。

但他不能停。

每一针落下，那一丝微弱的热流就沿着经脉走出一小段，像在黑暗的隧道里划亮一根火柴。毒素被逼退一寸。只有一寸。

够了。一寸一寸地推，三十六针推出三十六寸，足够在丹田——在核心区域——围出一小片净土。

他不需要治愈。他只需要活到天亮。

第二十四针刺入后腰命门附近时，他终于忍不住闷哼了一声。

这一针扎在了一条半死的经脉上，内力反冲的痛感像电击一样窜上脊椎。银针嗡地颤了一下。他看见针尖渗出一滴黑色的血珠——是毒素被逼出来了。

"有用……"他低声说，眼睛亮了。

最后十二针。

他换了长针。这十二针要更深，刺入的层次更接近……他不知道该怎么翻译，姑且叫"核心循环层"。前世他能把银针穿透十层宣纸而第十一层纹丝不动，现在他需要的精度比那更高。

呼吸放平。心跳放缓。

手停止了颤抖。

第二十五针。

第三十针。

房间里安静得只剩下针尖刺入肌肤时的细微声响和铜盆里热水偶尔晃动的声音。福叔连呼吸都放轻了，蹲在一旁，像守着一盏随时可能熄灭的灯。

第三十五针。

陆九针的视线开始模糊。不是因为泪水，是大脑供氧不足的前兆。还有最后一针。他把所有残存的意志拧成一股，集中在右手的拇指、食指和中指上。

最后一针，落在胸口膻中穴正下方一寸。

深入。旋转。留针。

一瞬间，三十六根银针同时一颤，像被某种看不见的力量串联起来。他感觉到一股热流——比之前任何一次都要明显——从三十六个针点汇聚，缓缓流向丹田。

毒素没有被清除。经脉没有被修复。

但丹田里那一星半点的内力，被他用三十六根银针织成了一张网，牢牢裹住了尚未扩散的毒素核心。

像急诊室里给大出血的病人做的临时填塞——不是根治，是争取时间。

他活下来了。

至少，今晚活下来了。

陆九针仰面倒在床上，胸口插着银针，全身是汗，大口喘着气。天花板上映着烛火的影子，晃晃悠悠。

"福叔。"他的声音轻得像一片羽毛，"我现在……经脉还剩多少？"

老人颤巍巍地伸手搭上他的脉。许久。

"不足……一成。"

一成。

十条车道只剩一条还能走。一百根电线只剩十根还通电。但还活着。还有脉搏。还有呼吸。还有这双手。

"够了。"他说。

窗外传来一声梆子响。三更天。他还有两三个时辰到天亮。天亮之后——

门开了。

不是推开的。是轻轻拨开的。像一个人不想发出任何声响，却又不在乎被发现。

一个年轻人站在门口。

二十五六岁，长衫玉冠，面如冠玉，嘴角挂着一丝笑意。那种笑容陆九针见过——急诊科门口等着领尸体的赌场打手，就是这么笑的。

"九针弟弟。"来人的目光掠过他胸口密密麻麻的银针，掠过床单上的黑色血渍，掠过蹲在一旁瑟瑟发抖的福叔，最终回到他脸上。

笑容没变。但眼睛里多了一样东西。

意外。

淡淡的、一闪而过的、不该出现的意外。

"这么晚了还没睡？"堂兄陆文远微微歪头，"弟弟脸色不太好，可是那碗燕窝粥不合胃口？"

陆九针看着他。

浑身三十六根银针，一成经脉，满身冷汗，半条命。

他笑了。

"粥很好。就是……太补了些。劳烦堂兄挂心。"

陆文远的目光在他身上停留了三息。

然后他转身离去，脚步声消失在走廊尽头。

门没关。

夜风灌进来，吹得烛火剧烈摇晃。陆九针盯着那扇空洞洞的门，笑意慢慢从脸上褪去。

他知道了。

燕窝粥是堂兄送的。断肠散是堂兄下的。而堂兄刚才来，是来确认他死没死。

他没死。

但堂兄现在也知道了。

天亮前，必须离开这里。

福叔还蹲在原地，嘴唇哆嗦着说不出话。陆九针低头看着自己胸口的银针——拔针至少还需要半炷香，拔完之后他能站起来走路，但仅此而已。一成经脉。在这个以武为尊的世界里，他和废人无异。

他闭上眼。

不怕。

前世急诊科三年，他在死神手里抢过几百条命。

这一次，先从自己的开始抢。
