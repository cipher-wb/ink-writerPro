# 禁忌与常见雷区

## 核心原则

- AI是写小说的增强工具而非救世主，需要掌握正确的使用技巧，但真正决定成功的仍是作者的核心创作能力和市场认知。
- 新人写小说应专注提升作品质量，无需依赖发红包、刷数据等虚假运营手段，真实读者追读才是决定成绩的关键
- 商业化创作需要深入研究并迎合大众情绪变化，在题材选择和主角设定上符合当前舆论偏好，才能提升作品的商业上限。
- 普通人在网文和自媒体行业的成功关键不在天赋，而在于坚持努力、充分发挥自身优点、虚心听劝改进。
- 新人作者应避免写传统玄幻、都市一兰、灵异、科幻等市场饱和或难度过高的题材，选择更有市场前景的题材以节省时间和成本。
- 新人作者要回归讲故事的本质，通过优秀的情节和节奏产生期待感而非依赖系统设定，同时学会分析优秀作品的写法来快速成长。
- 警惕虚构单章和夸大宣传，保持质疑精神识别真假信息，尊重创作客观规律通过长期积累而非捷径成功。
- 避免在前期过度消耗精华设定和读者热情，合理规划全书结构并维持创作动力，防止陷入成绩下滑→心态崩溃→断更烂尾的恶性循环。
- 全职写网文需要月稿费稳定破万、充足存款、持续产出能力和题材兜底绝技，普通人应先做兼职，未成年人应以学业为重。
- AI辅助创作是允许的，但要避免抄袭和过度依赖，通过融合学习而非复制粘贴来保持创作能力。
