# 综合建议

## 核心原则

- AI是写小说的增强工具而非救世主，需要掌握正确的使用技巧，但真正决定成功的仍是作者的核心创作能力和市场认知。
- 这是编辑星河小红书账号的内容索引，汇总了27期关于网文创作、运营、题材选择等多方面建议的视频内容目录。
- 以个人12年网文职业发展为例，强调写作对人生的意义和网文行业的财富上限，鼓励新人大胆尝试，呼吁社会给予创作者应有的职业尊重与认可。
- 警惕虚构单章和夸大宣传，保持质疑精神识别真假信息，尊重创作客观规律通过长期积累而非捷径成功。
- 网文作者可通过使用辅助软件、避免卡壳中断、学会章节拆分等实用方法来提升码字速度，同时保证写作质量。
- AI辅助创作是允许的，但要避免抄袭和过度依赖，通过融合学习而非复制粘贴来保持创作能力。
- 编辑通过直播审稿指出网文创作中的致命错误：人物设定混乱、身体数据离谱、角色命名不当、逻辑违和等问题严重破坏代入感。
- 网文作者需专注作品质量和真实读者，警惕虚假数据和过度依赖工具，同时要了解市场趋势和把握大众情绪。
- 探讨AI发展如何改变内容创作者在写网文、剧本和提示词上的创作思路和方法论
- 对比两位网文大师猫腻和烽火戏诸侯的创作风格和特点，帮助网文创作者选择学习对象和创作方向。
