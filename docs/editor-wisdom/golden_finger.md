# 金手指与主角能力设定

## 核心原则

- 通过反套路的主角人设（穿越成袁术但招义子而非沉迷美女）和权谋爽点的有机组合，打造了具有强市场竞争力的三国穿越题材作品，新人作品成绩飙升具有高商业价值。
- 新人写小说在金手指设定、题材选择和脑洞反差设计上需要的注意事项和创意思路
- 通过精心设计开头、人物身份、爽点场景和主副线平衡，精准吸引目标读者群体
- 网文创作需避免过度离谱的人物设定和不适当内容，严格遵守平台规范，网文平台不是法外之地。
- 末日文通过有智商的主角、自带禁忌感的女性角色、霸气的金手指设计和持续的冲突矛盾来制造爽感和张力。
- 新人作者需避免十大常见错误，重点简化开头、理性分卷、选择合适题材、虚心学习并稳健发展。
- 编辑通过评价低俗恶搞稿件，指出荒诞能力设定和低俗内容是平台审核禁区，整活文案无法获得签约。
- 系统惩罚流设定剧情闭环精妙富有吸引力，但主角行为涉及公共场所性骚扰等违法犯罪内容触犯法律底线，在主流平台过稿困难，建议投稿番茄等审核更宽松的平台。
- 金手指和开篇的核心是吸引读者、促进人物互动，而非遵循死板的黄金三章公式。
- 科幻与历史题材硬融合、金手指设定过度、节奏失控导致6章完结无法继续是编辑常见的一眼劝退稿件。
