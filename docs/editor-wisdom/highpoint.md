# 爽点与情绪高潮

## 核心原则

- AI是写小说的增强工具而非救世主，需要掌握正确的使用技巧，但真正决定成功的仍是作者的核心创作能力和市场认知。
- 新人开头应采用成熟故事模型加独特创新，通过营造低成本、高收益的'爽感'（身份反转、信息差等）来建立读者代入感和期待感，避免讲设定、流水账、上帝视角等常见错误。
- 通过强化主角人设与配角塑造、合理规划不同阶段章节卖点、多层次构建爽点升级与伏笔系统、以及维持稳定心态等七个技巧，构建长篇网文的稳定增长曲线并防止中期崩盘。
- 通过分析《娱乐1994》《当过奥特曼》《太平天国1854》等成功案例，揭示新人作者应选择具有独特职业切入点、小众历史垂类或自带流量的题材，以提高签约率和稿费收益。
- 新人应先掌握线性叙事建立代入感和故事骨架，再结合分镜式叙事控制节奏和爽点，根据题材灵活调整两种叙述方式的比例。
- 网文代入感需要从人物塑造、节奏把控、情感高点等多个维度综合打造
- 通过反套路的主角人设（穿越成袁术但招义子而非沉迷美女）和权谋爽点的有机组合，打造了具有强市场竞争力的三国穿越题材作品，新人作品成绩飙升具有高商业价值。
- 新人签约难的根本原因是开头阅读障碍高，应通过成熟故事模型降低成本、独特创新提供高收益爽感的"低成本、高收益"原则来吸引编辑和读者。
- 西幻新书要在第一章建立代入感、第二章通过隐藏主角实力拉期待感、第三章才讲设定，避免前期暴露天赋和强度，否则读者期待感会大幅下降。
- 通过精心设计开头、人物身份、爽点场景和主副线平衡，精准吸引目标读者群体
