# 人物塑造与角色设计

## 核心原则

- AI是写小说的增强工具而非救世主，需要掌握正确的使用技巧，但真正决定成功的仍是作者的核心创作能力和市场认知。
- 商业化创作需要深入研究并迎合大众情绪变化，在题材选择和主角设定上符合当前舆论偏好，才能提升作品的商业上限。
- 新人作者要回归讲故事的本质，通过优秀的情节和节奏产生期待感而非依赖系统设定，同时学会分析优秀作品的写法来快速成长。
- 新人作者通过研究市场、精选题材、听劝编辑、保持心态和稳定规划，可以在一年内实现月稿费从破万到七万的指数级增长。
- 编辑通过直播审稿指出网文创作中的致命错误：人物设定混乱、身体数据离谱、角色命名不当、逻辑违和等问题严重破坏代入感。
- 小说开头要在有限篇幅内快速传递核心信息和情感吸引点，避免无效铺垫和信息堆砌，通过道序、人物对比、情绪共鸣制造读者想继续阅读的欲望。
- 通过强化主角人设与配角塑造、合理规划不同阶段章节卖点、多层次构建爽点升级与伏笔系统、以及维持稳定心态等七个技巧，构建长篇网文的稳定增长曲线并防止中期崩盘。
- 成功作者需要自信、梦想和勇气敢于转型突破，主角的品质会反映作者的精气神与精力。
- 通过分析《娱乐1994》《当过奥特曼》《太平天国1854》等成功案例，揭示新人作者应选择具有独特职业切入点、小众历史垂类或自带流量的题材，以提高签约率和稿费收益。
- 网文代入感需要从人物塑造、节奏把控、情感高点等多个维度综合打造
