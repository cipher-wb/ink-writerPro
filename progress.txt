- Review gate: ink-writer/skills/ink-write/references/step-3-review-gate.md

- Tests: tests/ directory, pytest with pythonpath=. ink-writer/scripts ink-writer/dashboard scripts
- Review report JSON structure: overall_score (float), checker_results (dict→{overall_score, issues}), review_payload_json.entity_count, severity_counts
- Data-agent output has entities_appeared + entities_new for entity extraction counts
- Step 3 has 7 core checkers (always-on) + 6 conditional checkers
- Review bundle generated by: ink.py extract-context --format review-pack-json
- Slim bundle generation: slim_review_bundle.py --bundle X --checkers Y --outdir Z (per-checker field subsetting with fallback)
- Logic precheck: ink-writer/scripts/logic_precheck.py provides precheck_arithmetic() and precheck_attributes()
- Slim bundle precheck injection: slim_review_bundle.py --precheck flag runs logic_precheck and injects into logic-checker bundle
- Step 2B uses style-adapter.md, has dual-mode (full/targeted) based on step2b_metrics.py
- step2b_metrics.py: evaluate() returns {avg_sentence_length, dialogue_ratio, targeted_mode, long_sentences, summary_phrases}
- context-agent token budget: 8000(ch1-3) / 7000(ch4-30) / 9000(ch31-100) / 13000(ch101+), cap 15000
- data-agent.md is 792 lines, outputs entities/relations/summaries
- ai-word-blacklist.md at ink-writer/references/shared/ai-word-blacklist.md (49 lines)
