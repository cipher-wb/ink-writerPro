## Codebase Patterns
- YAML loaders 模式：参考 `ink_writer/reader_pull/config.py`，用 `Path(__file__).resolve().parent.parent.parent / "config" / "<name>.yaml"` 定位默认路径
- `open()` 全部带 `encoding="utf-8"`（CLAUDE.md Windows 兼容守则强制）
- ruff isort 格式：`import pytest` 与第一方 import 之间不留空行（同段视为一组）
- pytest.ini testpaths 单行追加新目录；不重复添加已有目录
- 项目无 mypy/pyright；typecheck 等价于 `ruff check` 通过
- `python3` 而非 `python`（macOS 默认 alias）
- ruff 默认开 UP017：`from datetime import timezone; timezone.utc` 必须改为 `from datetime import UTC; UTC`（Py 3.11+ alias）
- M3 期 evidence_chain 写盘：默认 base_dir 是 `Path("data")`；测试传 tmp_path 隔离
- LLM client stub 标准模式：`FakeLLMClient(responders=[...])` + `.messages.create(**kwargs).content[0].text`；FIFO 队列，超出列表时复用最后一个；调用 kwargs 记到 `.calls` 便于断言 prompt 内容（参考 `tests/writer_self_check/conftest.py`）
- prompt 模板中 JSON schema 示例的 `{` `}` 必须写成 `{{` `}}`，否则 `.format()` 会爆 KeyError
