## Codebase Patterns
- Python 3.12+ target; ruff for linting (pyproject.toml config), pytest with --no-cov for editor-wisdom tests (coverage config only covers ink-writer/scripts)
- Existing tests live in ink-writer/scripts/data_modules/tests/; new editor-wisdom tests go in tests/editor_wisdom/
- Import modules with hyphenated names via importlib.import_module() and sys.path manipulation
- Data source at /Users/cipher/Desktop/星河编辑/ has two subdirs: 编辑星河 (xhs) and 编辑星河_抖音 (douyin), total 287 .md files
- jsonschema package available for schema validation in tests; schema files live in schemas/ dir
- Rule extraction uses per-file-hash cache (rules_cache.json); dedup merges source_files across identical rule texts
- All .md files have a consistent format with # Title header on first line
- sentence-transformers + faiss-cpu needed for RAG; model BAAI/bge-small-zh-v1.5 for Chinese embeddings
- ink_writer/ is a new package dir; retriever lives at ink_writer/editor_wisdom/retriever.py
- Root CLAUDE.md and AGENTS.md are created by US-015; docs/editor-wisdom-integration.md has the full architecture doc
