# 末世/生存题材核心要素

## 核心矛盾

末世题材的核心不是"如何杀怪"，而是**资源稀缺下的人性选择**。

## 五大支柱

### 1. 资源管理系统

| 资源 | 稀缺度 | 冲突来源 |
|------|--------|---------|
| 食物/水 | 极高 | 抢夺、分配、保存、寻找 |
| 武器/弹药 | 高 | 制造、修复、交换 |
| 安全住所 | 中 | 防御、维护、迁移 |
| 医疗 | 极高 | 伤病无药、感染风险 |
| 信息 | 中 | 安全区、怪物规律、其他幸存者 |
| 人力 | 变动 | 信任、背叛、牺牲 |

### 2. 伦理困境设计

| 困境类型 | 示例 |
|---------|------|
| **电车难题** | 救A必须牺牲B，A和B都是重要的人 |
| **搭便车** | 有人想加入团队，但食物只够现有人员 |
| **背叛成本** | 盟友背叛了你，但你还需要他的技能 |
| **仁慈代价** | 放过敌人后对方带人报复 |
| **渐进堕落** | 为了生存一步步越过道德底线 |

### 3. 势力生态

- **军事型**：纪律严明但个人自由受限
- **商人型**：以物易物，利益驱动
- **宗教型**：信仰凝聚但可能走向极端
- **流民型**：无组织，弱肉强食
- **技术型**：试图重建文明，但资源消耗巨大

### 4. 节奏设计

| 阶段 | 核心 |
|------|------|
| **灾难爆发** | 混乱、求生、与家人/朋友失散 |
| **初步生存** | 找到临时避难所，组建小团队 |
| **资源争夺** | 与其他势力冲突，面对第一次伦理选择 |
| **秩序重建** | 尝试建立规则，内部矛盾爆发 |
| **终极抉择** | 是苟且生存还是冒险追求更好的未来 |

### 5. 常见陷阱

| 陷阱 | 问题 | 改进 |
|------|------|------|
| 主角外挂 | 独自碾压一切 | 需要团队，每个人有不可替代的作用 |
| 无限弹药 | 破坏资源稀缺感 | 每颗子弹都有账本 |
| 怪物升级 | 为了难度而升级，破坏设定 | 怪物能力固定，人类学会利用弱点 |
| 人性真空 | 所有NPC都是好人/坏人 | 好人在极端条件下也会做坏事 |
