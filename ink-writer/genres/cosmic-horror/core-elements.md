# 克苏鲁/恐怖题材核心要素

## 核心理念

恐怖来自**未知**和**无力感**，不是来自血腥和暴力。

## 五大支柱

### 1. 恐怖氛围营造

| 技巧 | 说明 | 示例 |
|------|------|------|
| **错位感** | 日常中出现不该存在的元素 | 镜子里的影子晚了0.5秒 |
| **递进感** | 恐怖逐步升级，不是一步到位 | 声音→影子→触感→实体 |
| **留白** | 最恐怖的是没有描述的部分 | "他看到了那个东西，然后再也无法开口说话" |
| **感官剥夺** | 限制信息获取增加不确定性 | 停电、浓雾、失聪 |
| **日常侵蚀** | 恐怖渗透到安全的日常空间 | 自己家的镜子、睡着的同事、熟悉的路 |

### 2. 规则体系设计

| 原则 | 说明 |
|------|------|
| **规则必须可学习** | 主角（和读者）可以通过观察推导规则 |
| **规则必须有例外** | 以为掌握的规则被打破，制造二次恐怖 |
| **规则有代价** | 利用规则需要付出代价（时间/健康/记忆） |
| **规则有层级** | 表层规则背后有更深层的规则 |

### 3. 心理崩溃递进

```
阶段1：怀疑（这不可能是真的）
  ↓
阶段2：确认（确实是真的，但可以解决）
  ↓
阶段3：挣扎（解决方案有代价，值得吗？）
  ↓
阶段4：选择（保全自己还是保护他人）
  ↓
阶段5：余波（幸存后的创伤和改变）
```

### 4. 信息管理

- **读者知道的 vs 角色知道的**：控制信息差制造悬念
- **不可靠叙述**：角色的感知可能被扭曲
- **线索投放**：前文埋线索，后文揭示更恐怖的真相
- **禁止信息倾泻**：规则说明要像角色自己发现的，不是百科全书

### 5. 爽点设计（恐怖题材特有）

| 类型 | 说明 |
|------|------|
| **规则破解** | 主角推理出规则并成功运用 |
| **惊险逃脱** | 差一点就死了的生还 |
| **真相揭示** | 碎片信息拼凑出完整真相的瞬间 |
| **反杀** | 利用规则反过来对付恐怖源 |
| **牺牲** | 有角色为他人牺牲，情感冲击 |
