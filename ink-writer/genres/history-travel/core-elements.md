# 历史穿越题材核心要素

## 核心矛盾

历史穿越的核心矛盾不是"如何发财"，而是**现代认知与古代规则的碰撞**。

## 五大支柱

### 1. 历史还原与虚构的平衡

| 原则 | 说明 |
|------|------|
| **大事不虚** | 重大历史事件（战争、政变、灾难）的时间线和结果不可篡改 |
| **小事不拘** | 具体人物互动、日常细节可以虚构 |
| **蝴蝶效应有限** | 主角的行为可以改变小事，但历史惯性会抵抗大事 |
| **认知边界** | 主角不是百科全书，只知道自己专业领域+常识级别的历史 |

### 2. 穿越者的生存逻辑

- **语言障碍**：古代语言与现代汉语差异巨大，至少需要适应期
- **身份融入**：户籍、身份、社会关系都需要合理解决
- **知识变现**：现代知识不是万能的，需要材料和工具链
- **心理适应**：卫生条件、价值观差异、孤独感

### 3. 年代感塑造

| 维度 | 要素 |
|------|------|
| **物质** | 衣食住行的具体细节（什么朝代用什么餐具） |
| **礼仪** | 称谓、礼节、禁忌（避讳、男女大防） |
| **观念** | 忠孝、门第、命运观、生死观 |
| **节奏** | 没有即时通讯，信息传递需要时间 |

### 4. 金手指控制

| 类型 | 合理度 | 限制方式 |
|------|--------|---------|
| 专业知识 | 高 | 仅限本专业，其他领域只有常识 |
| 历史预知 | 中 | 只知大事，具体日期和细节模糊 |
| 系统辅助 | 中 | 功能有冷却和代价 |
| 随身空间 | 低 | 容量有限，不能存活物 |

### 5. 节奏设计

| 阶段 | 章节范围 | 核心任务 |
|------|---------|---------|
| 生存期 | 1-50 | 解决身份、温饱、基本安全 |
| 发展期 | 51-200 | 利用知识积累资源和人脉 |
| 影响期 | 201-400 | 开始影响历史走向 |
| 抉择期 | 401+ | 面对"改变历史"的终极选择 |
