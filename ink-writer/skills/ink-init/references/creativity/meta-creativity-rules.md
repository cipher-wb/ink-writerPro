# 创意元规则库 Meta Creativity Rules（M01-M10）

> **定位**：本文件是 `/ink-init --quick` 生成 3 套方案时的**硬约束检验标尺**，任何一套方案若未命中 ≥3 条元规则，必须重新抽取种子与扰动对后再生成。
> **使用方式**：在 Quick Step 0 作为 L1 必读加载；在 Quick Step 1 生成方案后，对每套方案打分标注命中的 Mxx 编号。
> **与其他文件关系**：元规则是**抽象层**，`anti-trope-*.md` 是**题材实例层**，`anti-trope-seeds.json` 是**具体素材层**，`perturbation-engine.md` 负责**混搭机制**。

---

## M01 代价可视化（Cost Visibility）

**核心定义**：金手指/能力/资源的每一次使用，都必须在**同一场景内**呈现出读者可感知的代价（身体、关系、时间、道德、寿命、精神、记忆、命运之一）。代价不能只靠设定文字交代，必须用画面化的后果演出。

**正例**：
- 《诡秘之主》序列者每次越界仪式都会增加"失控"概率，直观体现为瞳孔、体温、饮食异变。
- 《全职法师》莫凡双系魔法早期被"双系需要两倍灵气"强行拖慢，练级画面必演饥饿和晕厥。

**反例**：
- 主角开挂后"略感疲惫，休息一晚即可恢复"——代价等同于没有。
- 系统提示"此次能力消耗 30% 能量"——数字代价无画面，读者麻木。

**适配题材**：仙侠 / 都市 / 末世 / 科幻 / 游戏 / 悬疑

---

## M02 反向冲突（Inverted Conflict）

**核心定义**：主角的核心冲突**不是**"弱者变强踩脸"，而是反转常规方向——强者被迫示弱、天才被迫平庸、复仇者被迫宽恕、救世主被迫摆烂。逆向才能撕开套路。

**正例**：
- 《赘婿》宁毅是现代金融专家穿越去做赘婿，冲突在于"必须藏拙"而非"立刻翻身"。
- 《不科学御兽》顾凡的御兽属性越强，他越要克制使用，以免暴露体质风险。

**反例**：
- 主角开局被辱，三章之内打脸，打脸完继续被辱再打脸——方向单一。
- 废柴觉醒极品天赋，从此一路无敌——冲突消解。

**适配题材**：都市 / 仙侠 / 历史 / 游戏 / 恋爱 / 现实

---

## M03 信息不对称（Information Asymmetry）

**核心定义**：主角与反派/盟友/读者之间**至少存在一层信息差**，并且这层差被故事持续利用（误会、伪装、情报战、读者全知视角）。信息平权等于没有戏剧。

**正例**：
- 《庆余年》范闲知母亲之死背后的秘密比对手晚了整整一卷，读者全程替他紧张。
- 《宰执天下》韩冈利用北宋官场的奏折时差做政治博弈。

**反例**：
- 主角一进场就知道所有隐藏任务和 NPC 身份——悬念清零。
- 所有角色互相没有秘密，遇事直接摊牌——失去张力。

**适配题材**：悬疑 / 历史 / 都市 / 科幻 / 游戏 / 恋爱

---

## M04 角色错位（Role Displacement）

**核心定义**：主角的**身份**与所处**场域**错位——用错误的身份做正确的事，或被迫用正确身份做错误的事。错位越大戏剧能量越大。

**正例**：
- 《剑来》陈平安骊珠洞天的贫穷少年，却要与神仙打交道，身份和场域全程错位。
- 《诛仙》张小凡是正道弟子却修魔教秘籍，内心撕裂贯穿全书。

**反例**：
- 皇子在皇宫继承皇位——身份场域完全匹配，戏剧强度归零。
- 修仙家族子弟修仙——零错位。

**适配题材**：仙侠 / 历史 / 都市 / 末世 / 游戏

---

## M05 时空错位（Temporal-Spatial Displacement）

**核心定义**：故事的核心冲突必须绑定一个**具体的时间/空间紧迫性**：倒计时、禁区、异域、跨代、轮回。"随时随地都能打"的冲突没有张力。

**正例**：
- 《三体》黑暗森林法则下的"威慑纪元"——时间即威慑，每一秒都在消耗。
- 《流浪地球》地球必须在 2500 年内抵达半人马座——空间位移=时间压力。

**反例**：
- 主角在一个"时间无限"的宗门内慢慢修炼，没有任何外部倒计时——读者缺乏追读钩。
- 冲突可以在任意地点任意时间发生，没有"必须现在""必须在这里"——戏剧感丧失。

**适配题材**：科幻 / 末世 / 悬疑 / 仙侠 / 游戏 / 历史

---

## M06 身份反差（Identity Contrast）

**核心定义**：主角对外表现的**社会身份**与其内在**核心身份**必须形成可视的反差（至少二层皮：表层身份 vs 隐藏身份）。反差越具体越好。

**正例**：
- 《美食供应商》袁州表面小饭店老板，实际隐藏料理天才+秘方继承人。
- 《凡人修仙传》韩立表面普通弟子，骨子里是最冷静的"凡人"。

**反例**：
- 主角是天才少年→表现得像天才少年→被所有人承认为天才——零反差。
- 身份和能力完全一致，没有表里差——人设扁平。

**适配题材**：都市 / 仙侠 / 悬疑 / 历史 / 恋爱

---

## M07 欲望悖论（Desire Paradox）

**核心定义**：主角的**表层欲望**（想要 X）与**深层需求**（真正需要 Y）必须相互矛盾。故事推进=不断逼近深层需求、不断远离表层欲望。

**正例**：
- 《鬼吹灯》胡八一想发财（表层），却一次次为了保住兄弟/守护秘密而放弃财富（深层）。
- 《亮剑》李云龙想打胜仗立功（表层），真正需要的是部下少死（深层）。

**反例**：
- 主角想变强→变强→继续想变强——单线欲望无张力。
- 表层欲望直接=深层需求，整本书只有"升级"二字。

**适配题材**：都市 / 历史 / 现实 / 仙侠 / 恋爱

---

## M08 系统反噬（System Backfire）

**核心定义**：金手指/系统/外挂本身必须具备**反噬/背叛/异化**的可能性，并在故事中真实兑现一次以上。工具性金手指=没有金手指。

**正例**：
- 《诡秘之主》克莱恩的"愚者"途径本身会诱导他走向"小丑"失控。
- 《灵境行者》灵境反向吞噬觉醒者，是恐惧具现化的双刃。

**反例**：
- 系统永远无条件服从主角，随叫随到——工具型挂。
- 签到 100 年，除了奖励没有任何副作用——反噬缺席。

**适配题材**：仙侠 / 都市 / 科幻 / 游戏 / 末世

---

## M09 权力颗粒度（Power Granularity）

**核心定义**：世界观中的**权力/秩序**必须有**可见的颗粒度**（阶级、职能、空间、仪式、资源、义务），而不是模糊的"大能"或"老祖"。颗粒度越细，戏剧越有抓手。

**正例**：
- 《亵渎》罗格教会的七重品阶 + 各教区的辖区制度，形成细密的权力网。
- 《宰执天下》宋代官场阶、勋、职、差遣四套标签并行，使政争有具体抓手。

**反例**：
- 整个门派只分"老祖/长老/弟子"三级，细节全无——权力扁平。
- 大势力与小势力之间没有中间层，冲突只能"灭门"——颗粒度失焦。

**适配题材**：仙侠 / 历史 / 末世 / 科幻 / 游戏

---

## M10 尺度跃迁（Scale Leap）

**核心定义**：每卷/每个大章节必须完成一次**叙事尺度的跃迁**——从个人到家族、从家族到城池、从城池到王朝、从王朝到天地。尺度不跃迁=故事停滞。

**正例**：
- 《诛仙》从草庙村→青云门→正魔大战→太古遗迹→六界——每卷尺度递增。
- 《全职高手》从个人退役→俱乐部→联赛→国家队——层层跃迁。

**反例**：
- 全书 200 章都在一个山村里打转，尺度从未突破——读者疲劳。
- 尺度跃迁只在结局仓促完成，前文没有铺垫——崩盘。

**适配题材**：仙侠 / 末世 / 科幻 / 历史 / 游戏

## 方案校验硬门槛

`/ink-init --quick` 生成 3 套方案后，对每套方案按以下规则打分：

1. 对每条 Mxx 判断：**命中 / 部分命中 / 未命中**。
2. 单套方案至少命中 **3 条以上**（含部分命中计半分，总分 ≥3）才算通过。
3. 3 套方案合计必须覆盖 **≥7 条元规则**（保证差异化）。
4. 未通过者丢弃并重新抽取扰动对、重新生成。
5. 生成方案时将命中的 Mxx 编号标注于方案末尾，字段名 `meta_rules_hit`。
