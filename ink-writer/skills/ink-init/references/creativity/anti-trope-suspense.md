# 反套路库：悬疑/灵异/脑洞

> 覆盖题材：悬疑脑洞、悬疑灵异、女频悬疑
> 用途：在 `/ink-init` Step 5 创意约束包生成时加载，主动预警套路陷阱
> 注意：规则怪谈/克苏鲁已有 `anti-trope-rules-mystery.md` 覆盖，本文件专注纯悬疑/灵异

## 高危套路清单

### S级套路（极度泛滥，强烈建议避免）

| 套路 | 问题 | 反套路方向 |
|------|------|----------|
| **主角直觉破案** | "我有种不好的预感"替代推理 | 每个结论必须有可回溯的线索链，读者有机会先于主角推理 |
| **凶手是最后出场的人** | 读者无法公平推理，揭示像作弊 | 凶手从第一幕就出现，读者回看时恍然大悟 |
| **灵异万能解释** | 推理到一半变成"鬼干的" | 即使有灵异元素，也要有内在规则和逻辑限制 |
| **受害者活该** | 被害者全是坏人，消解悬疑紧张感 | 至少一个受害者值得同情，增加破案的道德紧迫感 |
| **警察/侦探全知** | 主角一到现场就看出一切 | 主角有专长也有盲区，需要团队互补或从错误中修正 |

### A级套路（常见但可改良）

| 套路 | 改良方向 |
|------|---------|
| **暴风雪山庄** | 封闭空间OK，但要有不封闭时无法成立的核心诡计 |
| **多重人格** | 人格分裂有医学基础限制，不是万能工具 |
| **死亡讯息** | 死者留下的线索要有"为什么用这种方式"的合理性 |
| **不在场证明核心** | 不在场证明可以是迷雾而非核心，真正的诡计在动机层 |
| **连环杀手无动机** | 连环杀手有明确的选择逻辑和心理画像 |

## 悬疑爽点替代方案

| 传统爽点 | 替代方案 |
|---------|---------|
| 天才顿悟 | 读者和主角同步获得关键信息，顿悟是"连线"而非"凭空" |
| 反转再反转 | 反转有前置线索支撑，不是为反转而反转 |
| 凶手自白 | 凶手有自己的逻辑和立场，自白时读者理解（不认同）其动机 |
| 灵异显灵 | 灵异规则体系化，显灵有触发条件和限制 |
| 绝境逃生 | 逃生利用前文已展示的环境/物品/技能，不开新挂 |

## 悬疑公平性检查清单

- [ ] 核心线索在揭示前已展示给读者？（公平推理原则）
- [ ] 凶手/谜底不是最后才引入的角色/信息？
- [ ] 灵异元素有内在规则且规则一致？
- [ ] 主角的推理过程有可追溯的逻辑链？
- [ ] 动机合理且与凶手的人物画像一致？
- [ ] 至少有一个误导线索让读者走弯路？（增加参与感）
