# 语言风格三档分级（Style Voice Levels）

用于 Quick 模式 3 套方案的语言质感差异化。每次 quick 生成时，Quick Step 1.6 强制把 V1 / V2 / V3 各分配给一套方案，禁止三套同档。

---

## 一、档位定义

### V1 文学狂野（Literary-Raw）

- **定位**：意象粗粝但有美感，句式短促、动词锋利、名物化，允许通感与冷硬隐喻。对应严肃文学/新武侠/克苏鲁底色网文的审美。
- **句感关键词**：锈、灰、潮湿、裂口、铁腥、骨缝、雪崩、低烧、断句、沉默。
- **人名倾向**：单姓单名或复姓单名，字面冷硬（池冷灰 / 陆无归 / 裴断雪 / 沈锈 / 陆寂寒 / 崔蚀）。
- **书名倾向**：2-5 字硬物象 + 动势词，常带反讽／矛盾修辞。
  - 示例：《赤地一拳》《霓虹佛陀》《锈铁观音》《灰烬新娘》《白骨钟声》《哑炮》
- **粗口密度**：0.2%-0.4%（克制，多为 L0 中性俚语，偶用 L1，禁用 L2+）。
- **禁忌**：拒绝口水化解释、拒绝「哈哈哈」式情绪词、拒绝爽文打脸套话。

### V2 烟火接地气（Everyday-Smoky）

- **定位**：口语化、方言感、市井气息浓。适合都市/职场/现实/小人物逆袭题材，让读者「一秒进场」。
- **句感关键词**：咸淡、烟油、铁皮桌、塑料凳、地摊、电动车、外卖箱、楼道灯、早八、缓冲池。
- **人名倾向**：大俗姓 + 接地气字或重叠字（王满仓 / 赵二娃 / 李铁柱 / 刘艳红 / 陈阿嬷 / 孙小芳），允许「老 X」「阿 X」「小 X」前缀。
- **书名倾向**：第一人称 / 口语句 / 接地气动作 + 反预期目的地，常带双关或谐音。
  - 示例：《老子就是救世主》《外卖送到阎王殿》《我妈是万元户》《退休第一天我成了村长》《我在菜市场卖神仙》
- **粗口密度**：0.3%-0.7%（L0 放开，L1 常用：卧槽/妈的/他娘的/龟儿子，L2 禁用）。
- **禁忌**：拒绝高冷文艺腔、拒绝成语堆砌、拒绝西式翻译腔长定语。

### V3 江湖野气（Jianghu-Wild）

- **定位**：绰号感 + 轻度粗口 + 快刀剪裁的野路子文风。适合江湖、黑道、荒村怪谈、破案、末日等需要强烈「下作气场」的题材。
- **句感关键词**：刀疤、烂命、狠话、麻绳、血茬子、骨灰、破锣嗓、落草、三刀六洞、吐口痰。
- **人名倾向**：以绰号为主要称呼（刀疤阿九 / 臭嘴老七 / 瞎眼徐 / 独耳麻子 / 癞子三 / 二愣赵 / 闷瓜子 / 瘸脚福），真名为辅。
- **书名倾向**：口头禅式惊叹句 / 反套路俗语 / 粗粝动势词，允许谐音或反讽。
  - 示例：《卧槽这关没打过》《神仙打屁》《阎王少收我三年》《老子不伺候了》《狗日的修真界》《这狗命我不要了》
- **粗口密度**：0.8%-1.5%（L1 常用，L2 擦边在档位 3+ 允许，L3 全禁）。
- **禁忌**：拒绝玛丽苏/霸总腔、拒绝过度书面语、拒绝仙气飘飘形容词堆。

## 二、敏感词 4 级分类

| 级别 | 类型 | 示例 | 允许档位 |
|------|------|------|---------|
| **L0 中性俚语** | 日常口头禅，无冒犯含义 | 老子 / 丫的 / 瞎整 / 扯淡 / 瞎鼓捣 / 瘪三 / 日怪 | V1/V2/V3 全档允许 |
| **L1 轻度粗口** | 情绪词性质，无生理/族群攻击 | 卧槽 / 妈的 / 他娘的 / 龟儿子 / 狗日的 / 傻逼(降写作「SB」或同音替换) / 滚犊子 | V2 + V3 允许 |
| **L2 擦边暗示** | 感官撩拨 / 身体化但不具象 | 腰软 / 眼神勾魂 / 湿热气息 / 呼吸紊乱 / 耳尖发烫 | 仅 V3 + 激进档位（3/4）允许 |
| **L3 红线** | 具体生殖器官词 / 性行为描写 / 辱骂族群 / 宗教辱称 / 地域仇恨 | 略（不列示以免污染语料） | **全档禁止**，命中即重写 |

### 红线硬规则

- L3 命中一次 → 该套方案整体重写，不做「打码」或「替换」。
- L2 只能出现在 V3 且档位 ≥3（激进/疯批）；且每章 ≤3 处。
- 所有档位均禁止：性别羞辱词（如「母狗」系列）、未成年性暗示、地域族群嘲讽（地图炮）、具体生殖器官直写。

## 三、密度矩阵（粗口占比 = 粗口字数 / 正文总字数）

| 激进度档位 | 总粗口密度 | L0 上限 | L1 上限 | L2 上限 |
|-----------|-----------|--------|--------|--------|
| 档位 1 保守 | 0%（零容忍） | 0% | 0% | 0% |
| 档位 2 平衡 | ≈0.2% | 0.2% | 0.05%（偶发情绪峰值） | 0% |
| 档位 3 激进 | 0.5%-0.8% | 0.4% | 0.3% | 0.1%（仅 V3） |
| 档位 4 疯批 | 0.8%-1.5% | 0.6% | 0.6% | 0.3%（仅 V3） |

注：密度以「章」为窗口统计，单章溢出 1.5 倍视为违规。

## 四、Quick Step 1.6 分配算法

```
inputs: draft_1, draft_2, draft_3 (已通过 Quick Step 1.5 的三套方案)
        level ∈ {1,2,3,4}

1. 候选档位集合 = shuffle([V1, V2, V3])
2. 按种子（stable_hash(timestamp + genre_tuple)）固定顺序分配：
     draft_i.style_voice = 候选[i]   (i=1,2,3)
3. 标注每套方案 style_voice_level + density_target
4. 校验：
     - 禁止两套方案同档
     - 档位 1 保守 + 被分到 V3 时 → 强制降级到 V1（V3 仅档位 2+ 允许）
     - 档位 4 疯批必须至少一套 V3
5. 输出字段（schema）：
     {
       "style_voice": "V1|V2|V3",
       "density_target": "0.2%",
       "sample_line": "<5-15 字示例句>",
       "vocabulary_allowlist": ["L0", "L1"?, "L2"?]
     }
```

### 特例：档位 1 保守

档位 1 下 V3 不可用；若被分配到 V3，回退规则：V3 → V1（而非 V2，保留风格差异）。Quick Step 1.6 日志需标注 `style_fallback: "V3→V1 (level=1)"`。

### 特例：档位 4 疯批

三套方案中至少一套必须是 V3，否则重分配。疯批档 V3 粗口密度上限放宽到 1.5%，但仍受 L3 红线约束。

## 六、与 Layer 1/2/3 的协作

- **Layer 1 元规则**：本档位分级不改写 M01-M10 的任何约束；V3 出现 L2 擦边时仍需 M01 代价可视化。
- **Layer 2 种子库**：种子 `genre_tags` 不含风格档位，种子值本身可跨档使用，但 V1 避免俚语化种子表达。
- **Layer 3 扰动引擎**：扰动对注入不区分档位；注入后生成句子由本文档控制粗口密度与词表。
- **Quick Step 1.5 金手指校验**：金手指爆点句（GF-3）若是 V3 档位方案，允许用 L1 粗口提升「这也行？」惊讶感，但不得用 L2/L3。

---

## 七、校验硬门槛

- 3 套方案必须两两不同档位（V1 + V2 + V3 各一套）。
- 粗口密度超档位矩阵上限 → 该章重写，不做局部替换。
- L3 命中 → 方案整体重写（不允许保留部分段落）。
- Quick Step 2 创意指纹「语言档位」字段消费本文档的 `style_voice` + `sample_line`。

## 九、风格落地句式库

下列句式为 writer/polish-agent 起草/润色时可直接调用的语感模板，分档位给出 8 条典型骨架，用于保证档位特征在章节正文中可落地、可校验。

### V1 文学狂野句式

1. 「他把 X 按在 Y 上，像按一枚生锈的钉子。」
2. 「Z 的声音从墙后渗过来，带着潮气。」
3. 「她笑。笑里有铁。」
4. 「这场雪下到第三天，城里的灯全成了哑的。」
5. 「他听见自己的骨头在说话。」
6. 「风从裂口里出来，把夜吹瘦了一圈。」
7. 「刀不响。响的是他心里的门。」
8. 「那一口血吐在雪上，雪没有接。」

### V2 烟火接地气句式

1. 「卧槽，电动车又坏了，今天这单是黄了。」
2. 「阿姨，这葱多少钱一把？哎那我不要了。」
3. 「我妈打电话来，说让我回家吃饭，少废话。」
4. 「这破楼道灯一闪一闪的，跟我的工资条似的。」
5. 「他一屁股坐在铁皮桌边，掏出烟，烟盒空的。」
6. 「『哎，大哥，借个火呗。』『滚。』」
7. 「外卖箱往电动车上一挂，一脚油门，阎王爷都追不上。」
8. 「这日子过的，比我妈腌的咸菜还齁。」

### V3 江湖野气句式

1. 「老子这条命，不劳阎王费心。」
2. 「刀疤阿九一口痰吐在地上：『谁敢再说一句，我割他舌头。』」
3. 「那狗日的账，今天得清。」
4. 「『臭嘴，把酒拎过来。』『你自己没手？』」
5. 「他把刀拍在桌上，桌子裂了，人没动。」
6. 「瞎眼徐笑了，笑得像一口破锣。」
7. 「这帮狗娘养的，敢摸老子的箱子？」
8. 「三刀六洞他都挨过，这点事算个屁。」

## 十、常见跑偏与修复

| 跑偏症状 | 档位 | 修复动作 |
|---------|------|---------|
| V1 出现「哈哈哈」「嘻嘻」情绪词 | V1 | 删除，换冷动作或沉默（参见句式 3/4） |
| V2 出现长定语从句（「被时代抛弃的那个孤独灵魂」） | V2 | 拆成 2-3 个短句，替换成动作/物件 |
| V3 粗口集中堆在一段 | V3 | 均匀分布，整章间隔 ≥200 字再出现一次 |
| 书名带「神帝/天帝/至尊」 | V1/V2/V3 全档 | 强制替换（由 blacklist.json 拦截） |
| 人名命中「萧尘/林风/叶寒」组合 | V1/V2/V3 全档 | 强制替换（由 blacklist.json 拦截） |
| V2 用文言成语堆砌（「天命所归」「宿命难逃」） | V2 | 替换为白话短句 |
| V3 使用仙气飘飘形容词（「绝美」「惊艳」） | V3 | 替换为刀疤/烂命/狠话系词表 |
| 档位 1 却出现 L1 粗口 | 档位 1 全档 | 删除，不做替换（保守档零容忍） |

## 十一、Voice 一致性约束

- 单本书全程锁定一档：一旦 Quick Step 1.6 分配完成，后续章节 writer/polish 不得切换档位。
- 视角角色可以临时「借档」：如 V1 书里引入一个 V3 黑道角色，其对白允许使用 V3 词表，但叙述语仍为 V1。
- 档位迁移触发条件：仅限用户在 `/ink-init` 非 quick 模式显式申请；quick 模式锁定后不允许迁移。

## 十二、与去 AI 味规则的衔接

本档位规则与 `polish-agent` 去 AI 味规则协同生效：
- polish-agent 判定的「AI 味高发词」（如「不禁」「缓缓」「不由得」）在 V2/V3 档位权重加倍扣分。
- V1 档位允许保留部分书面语长句，但仍禁用 AI 高发套话。
- 三档统一禁用「第 XX 日」式时间标记开头（起点编辑打回核心信号）。
