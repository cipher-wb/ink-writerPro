# 反套路库：末世/废土/生存

> 覆盖题材：末世、废土生存、丧尸、灾变
> 用途：在 `/ink-init` Step 5 创意约束包生成时加载，主动预警套路陷阱

## 高危套路清单

### S级套路（极度泛滥，强烈建议避免）

| 套路 | 问题 | 反套路方向 |
|------|------|----------|
| **末世超市搬空** | 灾变后主角轻松获取无限物资 | 物资争夺是末世核心矛盾，获取任何资源都有风险和竞争者 |
| **丧尸只杀炮灰** | 主角团永远安全，丧尸只咬路人 | 主角团也有伤亡/感染风险，每次战斗都有真实代价 |
| **末世一天建基地** | 秩序崩塌后立刻重建，无适应期 | 基地建设需要漫长的信任积累、资源谈判、内部矛盾处理 |
| **末世圣母救全人类** | 主角无差别救人且不付代价 | 资源有限时必须做取舍，救一个可能意味着放弃另一个 |
| **异能秒解一切** | 觉醒异能后末世变游戏 | 异能有明确上限和代价，高等级丧尸/灾变能克制异能 |

### A级套路（常见但可改良）

| 套路 | 改良方向 |
|------|---------|
| **独狼主角** | 独狼是因为信任创伤，被迫组队后信任重建是主线之一 |
| **反派人渣化** | 反派在末世逻辑下有合理动机（如牺牲少数保多数） |
| **女角色花瓶** | 女角色有生存技能和独立目标，不是等待拯救的对象 |
| **物资无限** | 建立物资消耗系统，每次行动有资源成本 |
| **末世恋爱优先** | 感情线嵌入生存决策中（因为信任才组队，而非为了恋爱） |

## 末世爽点替代方案

| 传统爽点 | 替代方案 |
|---------|---------|
| 搬空超市 | 在竞争者中智取关键物资，其他留给对手做诱饵 |
| 碾压丧尸 | 用环境/陷阱/策略解决丧尸群，蛮力只是辅助 |
| 基地称王 | 通过解决内部矛盾和外部威胁逐步建立威信 |
| 异能觉醒 | 异能觉醒伴随身体/精神代价，需要在使用和保命间权衡 |
| 打脸人渣 | 在资源博弈中用更聪明的策略让对手自食其果 |

## 末世真实感检查清单

- [ ] 物资获取有真实的成本和竞争吗？
- [ ] 异能/金手指有明确的上限和代价吗？
- [ ] 人际关系建立在利益交换和信任积累上吗？
- [ ] 末世环境（气候/疫病/基础设施崩塌）有真实影响吗？
- [ ] 道德困境有灰色地带，不是简单的善恶二分吗？
