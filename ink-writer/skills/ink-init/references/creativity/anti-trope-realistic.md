# 反套路库：现实题材

> 覆盖题材：现实、职场、社会、家庭、生活流
> 用途：在 `/ink-init` Step 5 创意约束包生成时加载

## 高危套路清单

### 🔴 S级套路（极度泛滥）

| 套路 | 问题 | 反套路方向 |
|------|------|----------|
| **逆袭打脸** | 底层人物一路开挂打脸上层 | 逆袭有代价，每次上升都失去一些东西 |
| **圣母拯救** | 主角无条件帮助所有人 | 帮助有选择，有些人帮不了也不该帮 |
| **黑白分明** | 好人纯好，坏人纯坏 | 反派有合理动机，好人有自私时刻 |
| **金手指暴富** | 穿越/重生后凭记忆发财 | 发财有法律风险、伦理困境、能力边界 |
| **职场万能** | 初入职场就碾压所有人 | 专业能力强但社交/政治弱，或反之 |

### 🟡 A级套路（改良方向）

| 套路 | 改良方向 |
|------|---------|
| **底层奋斗** | 不是所有努力都有回报，展示结构性困境 |
| **家庭温暖** | 温暖中有真实摩擦和代际差异 |
| **职场宫斗** | 不只是人际斗争，展示行业生态和制度博弈 |
| **婚姻危机** | 不是出轨/小三的简单叙事，而是价值观渐行渐远 |

## 现实题材核心原则

1. **复杂性原则**：每个角色至少有一个"让读者理解其立场"的moment
2. **代价原则**：每个选择都有代价，没有"完美解"
3. **真实感原则**：细节决定可信度——具体的金额、具体的流程、具体的行业术语
4. **节奏感原则**：现实题材也需要爽点——"真相揭露""困局破解""小人物的高光时刻"
5. **社会性原则**：个人命运与社会环境交织，不是真空中的个人奋斗

## 现实题材的爽点设计

| 爽点类型 | 描述 | 示例 |
|---------|------|------|
| **真相揭露** | 隐藏的信息被公开，力量格局改变 | 年终审计揭露管理层的秘密账户 |
| **逆境翻盘** | 在绝对劣势中找到突破口 | 被裁员后用前公司忽略的专利创业 |
| **认知突破** | 角色突然理解了之前不懂的道理 | 理解父母当年选择背后的无奈 |
| **小人物高光** | 普通人在关键时刻展现非凡 | 快递员用路线知识帮警方找到嫌疑人 |
| **困局破解** | 看似无解的困境被巧妙化解 | 用规则漏洞合法维权，不是暴力对抗 |
