# 遣词造句工艺规则（Prose Craft Rules）

> 用于 writer-agent L10 遣词造句律执行参考，并作为 proofreading-checker Layer 6（Prose Craft Quality）与 polish-agent Layer 8（Prose Craft Polish）的量化判据。
> 核心原则：**强动词代替弱动词** + **感官锚点打破视觉沙漠** + **具象细节替代空洞形容词**。违反这三条是编辑 1-2 分文风评分的直接诱因。

## 一、弱动词黑名单（L10a 强动词法则）

### 黑名单 10 词

以下 10 个弱动词在章内使用频次**硬上限 ≤ 3 次/词**，超出即触发 proofreading-checker Layer 6 的 `WEAK_VERB_OVERUSE`：

| 弱动词 | 问题类型 | 全章阈值 |
|-------|---------|---------|
| 是 | 系动词判断，缺乏动作力 | ≤ 3 次 |
| 有 | 存在判断，信息贫瘠 | ≤ 3 次 |
| 做 | 泛化动作，无具体形态 | ≤ 3 次 |
| 进行 | 公文腔冗余 | ≤ 3 次 |
| 开始 | 起始状态描述，无实质 | ≤ 3 次 |
| 觉得 | 内心标签，跳过身体反应 | ≤ 3 次 |
| 感到 | 内心标签，跳过感知细节 | ≤ 3 次 |
| 看到 | 视觉过程词，无画面 | ≤ 3 次 |
| 听到 | 听觉过程词，无声音质感 | ≤ 3 次 |
| 想到 | 思维过程词，无具体念头 | ≤ 3 次 |

**全章总量**：10 词累计 ≤ 15 次。超出 → `WEAK_VERB_OVERUSE` critical。

**例外**：直接引语（角色台词）内部、短章末尾抒情段（≤ 100 字）不计入。

### 替换示例库（每词 5 场景 × ≥1 例）

#### 1. 「是」→ 具象动词/系表替换

| 场景 | 弱句（含"是"） | 强替换 |
|------|--------------|-------|
| 战斗 | 他是这条街最快的剑客 | 他的剑比街上任何人拔得都快 / 这条街没人能在拔剑上压过他半拍 |
| 日常 | 这是一个下雨的早晨 | 雨从天色未亮时就砸在瓦檐上 / 清晨的雨把窗棂敲得嗒嗒响 |
| 情感 | 她是他心里唯一的牵挂 | 她的名字压在他心口最沉的那一处 / 他每次呼吸都能触到她留下的重量 |
| 悬疑 | 那是一个不该存在的符号 | 那个符号根本不该出现在这张图纸上 / 符号的存在本身违背了所有档案记录 |
| 世界观 | 这是一个修仙的世界 | 修仙的律令编织着这方天地的每一寸经纬 / 天地的每一道纹理都浸着修仙的法则 |

#### 2. 「有」→ 具体动作/存在场景化

| 场景 | 弱句（含"有"） | 强替换 |
|------|--------------|-------|
| 战斗 | 他有三把剑 | 三把剑在他背后交叠成扇形 / 他腰间斜挂两把，背上还负着一把 |
| 日常 | 桌上有一杯茶 | 桌上一杯茶已凉得沉了底 / 茶汤在青瓷碗里静着，冒不出热气 |
| 情感 | 他心里有愧 | 愧意在他胸口坠了好几天 / 每次对上她的眼，愧就压住他的舌头 |
| 悬疑 | 房间里有股怪味 | 房间里浮着一股腥甜，像生肉放了三天 / 一种铁锈混着腐果的味道扑面 |
| 世界观 | 这个宗门有两万弟子 | 宗门名册压了两万条人命的重量 / 两万弟子的名字在执事堂的玉简里排成山 |

#### 3. 「做」→ 明确行为动词

| 场景 | 弱句（含"做"） | 强替换 |
|------|--------------|-------|
| 战斗 | 他做了一个反击 | 他反手一剑挑向对方手腕 / 剑尖从下而上削向对手的咽喉 |
| 日常 | 她做了一顿饭 | 她熬出一锅姜汤，煎了两个鸡蛋 / 锅里的米粥咕嘟了一整夜 |
| 情感 | 他做了一个决定 | 他咬碎嘴里那口血，点了点头 / 他把信揉成一团扔进炭盆，转身向外走 |
| 悬疑 | 他做了个手势 | 他竖起两根手指，指腹朝下 / 他在桌面上敲了三下，间隔不均 |
| 世界观 | 长老做了一个法事 | 长老在祭坛中央画下血符，诵起三段禁咒 / 长老九跪九叩，灯火在他肩头跳了七次 |

#### 4. 「进行」→ 删除或替换为动作

| 场景 | 弱句（含"进行"） | 强替换 |
|------|---------------|-------|
| 战斗 | 双方进行了激烈的对抗 | 剑光交缠了不下百招，血点溅在砖石上 / 两柄剑撞出三轮火星才各自退开 |
| 日常 | 他们进行了讨论 | 他们在灯下争到半夜，桌上的茶换了三壶 / 两人压着声音说话，话头拐了七个弯 |
| 情感 | 我们进行了深入的交流 | 我们把压了十年的话都翻出来晒 / 那晚她说了一夜，我听了一夜 |
| 悬疑 | 他进行调查 | 他把所有档案摊在桌上，一页一页翻过去 / 他把死者最后一周走过的每条巷子都走了一遍 |
| 世界观 | 宗门进行了秘会 | 九位长老在紫霄殿闭门三日未出 / 长老阁的门封了九重符印，殿内只留一盏长明灯 |

#### 5. 「开始」→ 直接写动作

| 场景 | 弱句（含"开始"） | 强替换 |
|------|---------------|-------|
| 战斗 | 他开始挥剑 | 剑光已扑到对方面门 / 他一剑劈下，不给对方拔刀的时间 |
| 日常 | 她开始煮饭 | 她往锅里添了三瓢水 / 米已下锅，她揭开灶膛续柴 |
| 情感 | 他开始哭 | 眼泪砸在他膝盖上 / 一滴湿意沿着他下颌线滚进衣领 |
| 悬疑 | 他开始怀疑 | 他把那份笔录又翻回第三页 / 他的目光停在签名那一栏，久久没挪开 |
| 世界观 | 天劫开始降下 | 第一道紫雷已劈开云层 / 天穹裂开一线暗紫，雷意压得呼吸都沉 |

#### 6. 「觉得」→ 身体反应/具体念头

| 场景 | 弱句（含"觉得"） | 强替换 |
|------|---------------|-------|
| 战斗 | 他觉得对手很强 | 对手的剑比他预想快了半拍 / 他手腕被震得发麻，才知道轻敌了 |
| 日常 | 他觉得累 | 他的肩膀垂下去，像被灌了铅 / 他坐在门槛上，一时没力气抬腿 |
| 情感 | 她觉得委屈 | 她嘴角抿着，眼眶红了两圈 / 她喉咙发紧，好久没说出话 |
| 悬疑 | 他觉得不对 | 他的后颈起了一层鸡皮疙瘩 / 一种说不清的凉意从脊背爬上来 |
| 修仙 | 他觉得丹田有股暖流 | 丹田深处一团热意在游走 / 暖意顺着经脉窜出三寸又缩回去 |

#### 7. 「感到」→ 感官锚点描写

| 场景 | 弱句（含"感到"） | 强替换 |
|------|---------------|-------|
| 战斗 | 他感到杀意 | 后颈的汗毛瞬间竖起 / 一股铁锈味的压迫感贴到他面门 |
| 日常 | 他感到饿 | 胃里空得发酸 / 饥饿在肚皮下咕咚咕咚地叫 |
| 情感 | 她感到失望 | 她垂下睫毛，手指慢慢松开了衣角 / 她抬头笑了一下，那笑却没抵达眼底 |
| 悬疑 | 他感到恐惧 | 恐惧从脚底往上爬，膝盖先软了半寸 / 他的牙齿咬出细微的咔嗒声，压不住 |
| 修仙 | 他感到灵力被压制 | 灵力到了胸口就卡住，像被一只手按着 / 丹田与经脉间好像隔了一层玻璃 |

#### 8. 「看到」→ 具体画面/动作承载视觉

| 场景 | 弱句（含"看到"） | 强替换 |
|------|---------------|-------|
| 战斗 | 他看到对手的破绽 | 对手右肩沉了半寸 / 对方剑势转向时，腰际露出一道缝 |
| 日常 | 她看到他回来了 | 门口一阵熟悉的脚步声 / 她抬头的瞬间，他的影子已落在门槛上 |
| 情感 | 他看到她的眼泪 | 一滴水顺着她下巴滑进衣领 / 她眼眶盛着光，光碎了就掉下来 |
| 悬疑 | 他看到墙上有血迹 | 墙角那一小片暗红在灯下发亮 / 灰墙上爬着一道干涸的暗痕，形状像个反写的"安"字 |
| 修仙 | 他看到一道剑光 | 一线寒芒已贴到他面门三寸 / 剑光劈空而来，空气被撕出刺耳的啸音 |

#### 9. 「听到」→ 声音质感描写

| 场景 | 弱句（含"听到"） | 强替换 |
|------|---------------|-------|
| 战斗 | 他听到身后的脚步声 | 碎石在身后被踏出一声脆响 / 沙沙的靴底摩擦声从三丈外贴近 |
| 日常 | 她听到敲门声 | 三下轻叩，间隔得很匀 / 门板被叩出两声闷响，像拳头而非指节 |
| 情感 | 他听到她在哭 | 压抑的抽气声从她喉咙里漏出来 / 一声极轻的呜咽，从隔门传来 |
| 悬疑 | 他听到奇怪的声音 | 一种类似指甲刮木头的细响从墙那边传来 / 像是有人在远处拖着铁链走路 |
| 修仙 | 他听到钟声 | 十二下钟声从青霄峰传来，尾音压了三息不散 / 钟声撞进他丹田，灵力都颤了一下 |

#### 10. 「想到」→ 具体念头/回忆画面

| 场景 | 弱句（含"想到"） | 强替换 |
|------|---------------|-------|
| 战斗 | 他想到师父教的招式 | 师父那一剑的轨迹在他脑中闪过 / 他手腕一翻，师父当年握着他腕骨画过的弧线又出现了 |
| 日常 | 他想到要去买菜 | 菜篮子昨夜就空了 / 灶台上那块腊肉只剩半指厚，今天必须下山一趟 |
| 情感 | 她想到他 | 他上次站在门口的模样又晃进她脑子里 / 他转身那一下，她又看了一遍 |
| 悬疑 | 他想到案子 | 死者指尖那道划痕又回到他眼前 / 他脑中反复放那三个时间点：九点十七，九点二十一，九点二十九 |
| 修仙 | 他想到当年的恩怨 | 三百年前那一剑劈碎山门的画面忽地清晰 / 师兄当年临死前那句话又敲进他心口 |

## 二、感官锚点规则（L10b 感官锚点法则）

### 核心要求

- **密度硬下限**：每 800 字至少 1 处**非视觉感官描写**（听觉/嗅觉/触觉/味觉/体感），纯对话段除外（对话≥ 80% 的段落豁免）
- **全章硬下限**：非视觉感官描写 ≥ 3 处（proofreading-checker Layer 6 `SENSORY_DESERT` 判据）
- **连续 800 字零非视觉感官** → `SENSORY_DESERT` high

### 四类非视觉感官示例

**听觉**：脚步声、呼吸声、风声、虫鸣、金属碰撞、远处钟声、纸张翻动、衣料摩擦

**嗅觉**：血腥味、松木香、焦糊味、药草味、汗味、花香、土腥、老屋的霉味

**触觉**：掌心的温度、冷汗、布料粗粝、石阶的凉意、风拂过后颈、剑柄的纹理

**味觉**：喉头的铁锈味（受伤）、舌尖发苦（紧张）、嘴里的茶涩、药味

**体感**：心跳加速、呼吸急促、肌肉紧绷、胃里翻涌、脚底发软、太阳穴抽痛

### 按场景配伍（参考 anti-detection-writing.md 的感官组合）

- **紧张/战斗**：优先触觉（剑柄纹理、掌心汗）+ 嗅觉（血腥、铁锈） — 求生感官
- **放松/日常**：优先味觉（茶香、饭菜）+ 听觉（远处脚步、檐雨） — 安全感官
- **受伤/虚弱**：某一感官失灵（视觉模糊/听觉闷塞）+ 另一感官放大（痛感/冷感）
- **情感场景**：优先体感（心跳、呼吸）+ 嗅觉（某人熟悉的气息） — 情绪感官

## 三、具象名词法则（L10c）

### 核心要求

- **空洞形容词堆叠禁区**：禁止连续 2 个以上空洞形容词修饰同一名词（如"美丽动人的""冰冷残酷的"叠加）
- **3 个及以上叠加** → proofreading-checker Layer 6 `ADJECTIVE_PILE` medium

### 空洞形容词示例（需精简或替换）

- **美丽的 / 漂亮的 / 迷人的** → 换为具体特征（"她左脸颊有颗小痣""她的眉比柳叶还细"）
- **强大的 / 可怕的 / 恐怖的** → 换为行动展示（"他一拳打穿三堵墙""那东西走过之处草木皆枯"）
- **邪恶的 / 冷酷的 / 残忍的** → 换为具体行为（"他擦净刀锋才看尸体一眼""他把俘虏留给乌鸦"）
- **神秘的 / 奇怪的 / 诡异的** → 换为具体细节（"那符号每一笔都逆时针写成""房里所有钟都停在三点十七"）

### 空洞感官词黑名单（proofreading-checker Layer 6 `GENERIC_DESCRIPTION` 判据）

以下空洞感官表述在章内出现 → medium 警告，需改为具体感官细节：

- 「气氛紧张」→ 具体化："空气像被抽走了一半""没人咳嗽，没人挪脚"
- 「感觉不对」→ 具体化："他后颈的汗毛竖了起来""脚下地砖的花纹忽然乱了一格"
- 「有种说不出的感觉」→ 具体化：直接写那种感觉的身体反应
- 「非常震撼」→ 具体化：写震撼带来的呼吸/姿势/言语变化
- 「无比悲伤」→ 具体化：写悲伤时角色做了什么（坐了多久/手指攥紧了什么）
- 「十分美丽」→ 具体化：一个能让人记住的独特细节
- 「空气凝固了」→ 具体化：谁先停下了什么动作
- 「时间仿佛静止」→ 具体化：具体哪几秒发生了什么
- 「莫名其妙的」→ 具体化：给出具体的反常点
- 「不可思议」→ 具体化：给出让人不可思议的那个事实

## 四、使用说明

### 对 writer-agent（L10 预防）

1. **写作前加载本文件**：Step 2A 执行前必读，与 `core-constraints.md`、`anti-detection-writing.md` 并列
2. **动笔即检查**：每写完一段，扫一眼本段是否连用 2 个以上弱动词；若有，立即用替换示例库改写
3. **感官密度节拍**：写到 800 字节点时主动检查，若前段全是视觉 → 下一段必插一个非视觉感官锚点
4. **形容词克制**：能用一个具象细节代替三个形容词时，坚决用细节

### 对 proofreading-checker（Layer 6 检测）

1. **逐词扫描弱动词**：对黑名单 10 词 `grep` 计数，任一词 > 3 → high，总计 > 15 → critical
2. **感官沙漠扫描**：滑动 800 字窗口检测非视觉感官出现次数，连续 800+ 字 0 次 → high
3. **空洞感官词扫描**：匹配"气氛紧张""感觉不对"等模板化表达 → medium
4. **形容词堆叠扫描**：正则检测同一名词前 ≥3 个修饰语 → medium

### 对 polish-agent（Layer 8 兜底）

1. **弱动词替换**：扫描全文，对超阈值的弱动词逐个替换为强动词（参考本文件示例库），保持语义不变
2. **感官锚点补充**：扫描感官沙漠区，插入 1-2 个自然的非视觉感官句（不破坏节奏）
3. **形容词精简**：扫描形容词堆叠，保留最精准的 1 个 + 补充 1 处具象细节
4. **空洞感官词改写**：逐条改写为具体身体反应/场景细节

## 五、与其他铁律的关系

- **L7 卖点落地律**：大卖点爆发段尤其禁用弱动词；反应段必须有非视觉感官（围观角色的倒抽气、颤抖、僵直）
- **L8 主角能动律**：主角主动行为优先用强动词；避免"他开始采取行动""他做了一个决定"这类弱动词句式
- **L9 人类本能反应律**：本能反应段（≥50-100 字）天然是感官锚点富集区，L10b 与 L9 互为支撑
- **scene-craft-index.md**：战斗场景的触觉+嗅觉、情感场景的小物件承载情感，都与本文件的感官锚点规则互通

## 六、例外与边界

- **第一人称内心独白**：允许少量"我觉得""我感到"作为视角锚定词，但仍需在 3 次阈值内
- **纯对话章节**：感官密度可降为每 1200 字 1 处非视觉感官（对话自身承担节奏）
- **短章收尾抒情段**（≤ 100 字）：弱动词计数豁免，但仍需至少 1 处具象细节
- **题材特殊性**：修仙章节可用"灵力涌动""元气充盈"等专门词汇；都市章节偏向生活感官；科幻偏向机械/电子感官
