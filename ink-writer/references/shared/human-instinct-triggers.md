# 人类本能反应触发词表

> 用于 writer-agent L9 人类本能反应律执行参考，以及 Context Contract 中 `本章人类本能触发场景` 字段填写参考。
> 核心原则：**有触发场景 → 必须有本能反应**。违反常识的无反应会让读者跳戏（"看到人倒下居然没人去扶"）。

## 一、五大触发场景分类

### 场景 A：有人受伤/倒下/求救

**触发关键词**：受伤、流血、倒地、摔倒、昏迷、呼救、求救、惨叫、濒死、中毒、中箭

**必须反应**（至少命中一项）：
- **在场主角/配角**：冲过去 / 蹲下查看 / 拨打求助 / 呼喊他人 / 施加救援动作
- **旁观群众**：惊呼 / 躲避 / 围观 / 拍照 / 议论
- **反应段≥80字**：描写反应的肢体动作/声音/情绪，不能一句"他愣了一下"带过

**允许的非典型反应**（需角色设定集中有对应标签）：
- `cold_blooded`（冷血）→ 可以麻木观察
- `mission_first`（任务优先）→ 可以选择忽略去执行任务
- `traumatized`（心理创伤）→ 可以僵住无反应
- `non_human`（非人类）→ 可以按设定的非人逻辑行动

### 场景 B：生命威胁（危险临头）

**触发关键词**：剑指咽喉、枪口对准、即将坠落、爆炸将至、被追杀、被围攻、刀架脖子、杀气迫近

**必须反应**（恐惧/紧张/战斗本能三选一+）：
- **恐惧反应**：手心出汗 / 呼吸急促 / 瞳孔收缩 / 腿软 / 冷汗顺着脊背
- **紧张反应**：肌肉绷紧 / 目光锁定对手 / 呼吸压低 / 心跳加速到能听见自己脉搏
- **战斗本能**：瞬间进入战斗姿态 / 下意识格挡 / 扫描逃生路线 / 寻找武器/掩体
- **反应段≥60字**：至少一个身体感官+一个思维/决策，不能只写"他很紧张"

**允许的非典型反应**：
- `battle_hardened`（身经百战）→ 反应更冷静但仍需有微表情/微动作
- `supernatural_calm`（异能/修炼者）→ 可以表现镇定但需内心波动或境界感
- `death_wish`（求死）→ 可以平静面对但需情感逻辑铺垫

### 场景 C：极端事件（震惊/应激场景）

**触发关键词**：亲眼目睹死亡、巨大爆炸、世界崩塌、至亲出事、家园被毁、突发灾难、见到鬼神、规则被打破

**必须反应**（震惊/应激）：
- **震惊表现**：瞳孔涣散 / 呆立原地 / 失语数秒 / 脑中一片空白 / 手中物件滑落
- **应激表现**：嘶吼 / 崩溃 / 颤抖 / 不受控的动作（后退/跌倒/掩面）
- **后续代价**：极端事件后的心理余波（夜不能寐/PTSD式闪回/短暂恍惚），至少一处后续段落体现
- **反应段≥100字**（比场景 A/B 更重）：因为是极端级别，不能一笔带过

**允许的非典型反应**：
- `witnessed_worse`（见过更惨）→ 可以冷峻但需有具体过往参照
- `reborn_veteran`（重生/老兵）→ 可以早有预判但需压抑感

### 场景 D：被侮辱/冒犯

**触发关键词**：当众羞辱、踩踏尊严、辱骂、嘲讽、扇耳光、嘲笑出身/能力、诋毁至亲、破坏信仰

**必须反应**（愤怒/隐忍/反击三选一）：
- **愤怒反应**：攥拳 / 咬牙 / 目光变冷 / 声音变沉 / 额角青筋
- **隐忍反应**：深呼吸 / 强压怒火 / 面无表情但手指颤抖 / 沉默转身
- **反击反应**：立即回怼 / 动手 / 用实力打脸 / 计入账本日后清算
- **禁止**：被侮辱后像什么都没发生，继续谈笑风生（除非设定为超脱境界）

**允许的非典型反应**：
- `transcendent`（超脱境界）→ 可以云淡风轻但需境界感体现
- `long_game`（长线布局）→ 可以表面顺从但内心明确记账

### 场景 E：获重大好处/机缘

**触发关键词**：捡到宝物、获得传承、获赠重金、被大人物赏识、能力突破、修为暴涨、意外机缘

**必须反应**（兴奋/谨慎/怀疑三选一+）：
- **兴奋反应**：呼吸急促 / 心跳加快 / 强压嘴角 / 握紧拳头 / 血液沸腾感
- **谨慎反应**：检查真伪 / 扫视四周是否有陷阱 / 追问来源 / 留后手
- **怀疑反应**：反常必有妖的第一反应 / 对方动机审视 / 自我怀疑是否做梦
- **禁止**：面对重大机缘毫无情绪波动（除非设定为见过更大世面）

**允许的非典型反应**：
- `seen_bigger`（见过更大世面）→ 可以平静但需对比感
- `burden_avoidance`（因故拒绝机缘）→ 可以拒绝但需充分动机

## 二、题材扩展触发场景

### 仙侠/玄幻扩展
- **渡劫失败/走火入魔**：场景 C 级别震惊 + 境界感知
- **灵石争夺/法宝炸炉**：场景 B 级别威胁 + 修士心性
- **宗门被屠/道侣陨落**：场景 C 级别极端事件
- **功法奥义顿悟**：场景 E 级别机缘 + 天道感应
- **天劫临头**：场景 B 级别威胁 + 境界反应

### 都市/现代扩展
- **职场霸凌/当众被贬**：场景 D 级别侮辱
- **车祸/突发病倒**：场景 A 级别受伤
- **家人出事/家庭变故**：场景 C 级别极端事件
- **意外中大奖/天降百万**：场景 E 级别机缘
- **被黑社会/反派盯上**：场景 B 级别威胁

### 科幻扩展
- **星舰被击中/生态舱破损**：场景 B 级别威胁 + 紧急协议
- **AI觉醒/失控/反叛**：场景 C 级别极端事件
- **虫洞/异常现象突发**：场景 C 级别震惊
- **获得外星科技/基因升级**：场景 E 级别机缘
- **外星人/怪物袭击**：场景 B 级别威胁 + 战斗本能

## 三、使用说明

1. **Context Contract 字段填写**：writer-agent 在写作前，从本章节拍/大纲中识别出将出现的触发场景，按"场景类型 + 关键触发事件 + 所需反应级别"格式填入 `本章人类本能触发场景` 字段。示例：
   ```
   - 场景B（生命威胁）：主角被反派剑指咽喉 → 紧张反应（≥60字身体感官+决策）
   - 场景E（重大机缘）：主角获得上古传承 → 兴奋+谨慎（反常必有妖）
   ```
2. **未触发场景跳过**：若本章无任何触发场景，该字段可填 `无`，但需在 Context Contract 中明确。
3. **反应段字数最低要求**：场景 A/B 各 ≥60-80 字，场景 C ≥100 字，场景 D/E ≥50 字。短于此值视为"一笔带过"（OOC-checker 会检测）。
4. **非典型反应的设定集标签**：若要使用非典型反应，角色设定集中必须有对应标签（如 `personality_tags: [cold_blooded, battle_hardened]`），否则视为违反人类本能反应律。
5. **围观群众反应不可省略**：当场景 A/B/C 发生在公共场所时，围观群众的反应也必须有体现（哪怕只有一两句），否则世界显得空洞。
