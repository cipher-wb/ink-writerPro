# AI高频词替换表

> 基于112本起点标杆小说 vs ink-writer产出的词频对比。
> 不是禁用词，而是"当使用频率明显高于标杆时触发替换建议"。
> polish-agent 在润色阶段扫描此表。

---

## 使用规则

- 当某个AI高频词在单章中的使用密度超过标杆均值的 **2倍** 时，标记为需替换
- 替换时优先选择"替换方向"列的具体写法
- 不是每次出现都要替换——偶尔使用是正常的，过度使用才是问题

## 替换表

| AI高频词 | 标杆频率(/万字) | 问题 | 替换方向 |
|---------|---------------|------|---------|
| 缓缓 | 0.9 | 动作不具体 | 写具体速度（"一寸一寸抬起头""慢到能数清每一根手指的动作"） |
| 微微 | 1.2 | 程度模糊 | 写具体程度（"嘴角刚弯了个弧度""肩膀几不可察地绷了一下"）或直接删掉 |
| 不由得/不禁 | 0.8 | 多余的修饰前缀 | 直接写反应，删掉前缀（"他笑了"比"他不由得笑了"更干脆） |
| 似乎/仿佛 | 2.5 | 过度使用显得犹疑 | 如果角色确定，直接写。如果不确定，用"好像"（更口语） |
| 一股XX涌上心头 | 0.3 | 万能情感模板 | 写具体身体反应（"胸口发紧""喉咙发酸""手心出汗""后背一凉"） |
| 目光扫过/落在 | 1.0 | 写"目光"而非写"看到什么" | 直接写看到的内容（"桌上的茶杯碎了"比"目光落在碎裂的茶杯上"好） |
| 深吸一口气 | 0.7 | 万能应激反应 | 换其他反应（攥拳、咬牙、后退一步、摸脖子、揉眼睛） |
| 意味深长 | 0.3 | 告诉读者"这句话有深意" | 写表情或动作（"他笑了一下，没接话""她端起茶杯喝了一口，视线落在别处"） |
| 心中暗道 | 0.5 | 内心独白标记词 | 直接写想法，不需要"暗道"前缀 |
| 眉头微皱 | 0.8 | 万能困惑/不满表情 | 换具体表情（"额头挤出三道纹""嘴角往下撇""眯了眯眼"） |
| 不自觉地 | 0.4 | 同"不由得"，多余前缀 | 直接写动作 |
| 这一刻/此刻/那一刻 | 1.5 | 时间标记过密 | 大部分可以直接删掉，动作本身就说明是"此刻" |
| 淡淡地 | 0.6 | 情感平淡化 | 要么写具体的"淡"（"声音平得像念菜单"），要么直接删掉 |
| 若有所思 | 0.3 | 告诉读者"他在想" | 写他做了什么（"手指无意识地敲桌面""视线飘向窗外"） |
| 显然 | 1.5 | AI式断言 | 大部分可删掉，让读者自行判断"显然"与否 |

## 使用频率参考基线

基于112本标杆小说统计（单位：每万字出现次数）：

| 词汇 | 标杆均值 | ink-writer均值 | 比值 |
|------|---------|---------------|------|
| 似乎 | 4.7 | 1.3 | 0.3x |
| 显然 | 1.8 | 0.9 | 0.5x |
| 一股 | 1.8 | 0.7 | 0.4x |
| 缓缓 | 0.9 | 0.6 | 0.7x |
| 微微 | 1.2 | 0.8 | 0.7x |

> 注：ink-writer实际AI词使用率低于标杆（标杆0.0030 vs ink-writer 0.0013），说明AI味的主要来源不是词汇，而是句式结构。此表主要用于polish-agent做精细化替换。
