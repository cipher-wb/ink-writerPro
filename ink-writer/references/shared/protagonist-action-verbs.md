# 主角行动动词表

> 用于 ink-plan Step 6 `主角行动` 字段填写参考。动词开头，≤30字，描述主角改变局面的主动行为。

## 被动描述黑名单（hard fail）

以下动词/动词前缀开头的描述视为被动，**不得用于 `主角行动` 字段**：

| 黑名单动词 | 原因 |
|-----------|------|
| 被 | 被动语态，主角非发起方 |
| 发现 | 观察行为，非主动改变 |
| 意识到 | 内心认知，非外部行动 |
| 感受到 | 感官体验，非主动决策 |
| 想到 | 内心活动，非改变局面 |
| 回忆 | 内心活动，无外部影响 |
| 观察 | 被动信息收集 |
| 注意到 | 被动感知 |
| 看到 | 被动感知 |
| 听到 | 被动感知 |

## 基础通用动词（所有类型适用）

### 战斗/对抗类
- 冲向、反击、对抗、拦截、突围、伏击、牵制、格挡、偷袭、挑战

### 决策/选择类
- 决定、选择、拒绝、牺牲、放弃、赌上、交换、妥协、坚持、背叛

### 社交/策略类
- 说服、欺骗、威胁、谈判、结盟、挑拨、揭穿、嘲讽、收买、警告

### 行动/执行类
- 使用、激活、破坏、窃取、逃离、伪装、潜入、搜索、调查、追踪

### 保护/拯救类
- 保护、拯救、掩护、救出、挡住、承受、替代、接住、背起、拖走

### 创造/获取类
- 建造、制造、收集、夺取、占领、解锁、升级、吸收、融合、召唤

## 仙侠/玄幻扩展

### 修炼/突破类
- 御剑、凝气、淬体、炼丹、布阵、渡劫、突破、吞噬、炼化、祭炼

### 战法类
- 施展、催动、引爆、封印、镇压、斩杀、驱散、吞噬、逆转、焚毁

### 门派/宗门类
- 挑战（长老/掌门）、闯关、夺旗、抢夺（秘境名额）、叛出、创立

## 都市/现代扩展

### 商战/职场类
- 撕毁（合同）、砸（场子）、谈判、收购、揭发、举报、跳槽、创业

### 人际/情感类
- 告白、分手、和解、道歉、对质、摊牌、公开（关系）、隐瞒

### 日常行动类
- 报警、拨打、搬家、跟踪、偷听（主动行为）、录音、拍照取证

## 科幻扩展

### 技术类
- 黑入、破解、编程、改造、启动、关闭、超频、重启、植入、提取

### 太空/探索类
- 起航、降落、对接、弹射、跃迁、扫描、登陆、采集、部署、回收

### 生化/改造类
- 注射、改造（自身）、进化、融合、分离、克隆、解毒、净化

## 使用说明

1. **优先从基础通用表选取**，除非场景需要类型专属动词
2. **白名单不穷举**：未在列表中但属于主动行为的动词均可使用
3. **判断标准**：该行为是否由主角主动发起、是否对局面产生改变
4. **连续重复预防**：连续 2 章使用同一类别的动词 → medium warning
