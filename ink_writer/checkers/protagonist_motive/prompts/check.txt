# 评估维度

1. **resonance（情感共鸣度）**：动机是否能让普通读者代入或理解（恐惧、爱、复仇、
   守护、不甘等普世情感）。空洞的"为了变强而变强"、"突然觉醒使命"、纯口号式宣言扣分。
2. **specific_goal（具体目标）**：是否给出一个**可被场景化**的近期目标
   （例如"找到失踪的妹妹"、"在雪季前修好祖屋"、"赎回母亲卖掉的剑"），而不是
   "称霸天下""万道归一"这类抽象远方目标。无具体目标扣分。
3. **inner_conflict（内在矛盾）**：动机内部是否有张力或代价
   （例如"想救人但救人会暴露身份"、"渴望和平却必须先卷入战争"），让选择本身有
   纠结感。如果只是单向行动、无任何阻力或代价，扣分。

# 输出要求

严格输出一段 JSON 对象（不要包裹在 markdown 代码块里），形如：

{{
  "resonance": <0.0~1.0>,
  "specific_goal": <0.0~1.0>,
  "inner_conflict": <0.0~1.0>,
  "notes": "<≤ 80 字中文说明，指出最弱的一两个维度>"
}}

约束：
- 3 个维度均为 0.0 到 1.0 的浮点数。
- 只在动机情感真挚、目标具体、内部有张力时才整体给 ≥ 0.85。
- 不要输出除 JSON 对象之外的其他文本。
